This is a small project of mine I started on March 20, 2026 for a community I'm apart of. Because I need a website to host this and I don't want to make a web application, I decided to use GitHub as the host.
It's all in one file, index.html, but everything works just standard HTML, CSS, and JS using MetroUI as the CSS framework (however I regret using it entirely for this).

Unlike my last project for this same community (the lapd call logger) this is a full indepth 1-1 system based on Motorolla's PremierOne MDT. Well... I say 1-1 but only applicable information is in here anything that
didn't fit into what we needed was removed. Besides that from all the references I could find, this should be a faithfull recreation of their MDT with the only extra thing being the sounds.

With that being said, consider this open source, use it as you want for your own projects as PremierOne is used in many enviroments and having this might be a good base to work with styles and functionality wise.
I can't promise it's eaisley configurable I even have some trobule fixing things lol

hope you enjoy!
